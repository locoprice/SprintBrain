Inline overlay · fires on trigger

overlay.css · #sprintbrain-overlay

Injected into any host page (Airbnb, WhatsApp, Gmail, etc.) when the user types a shortcut with required fields. Chrome is minimal so it reads as a polite visitor, not a takeover — all styles are !important-fenced in the real CSS to beat host styles, but the tokens are unchanged.
